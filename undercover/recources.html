Graphics

Each image on the Style Codes (Dating UC) page was created from scratch. The style code breakdown and tables were created in Photoshop.

Each banner’s text was also created from scratch using a custom font and Photoshop.

Embedded Media

On the Autumn Winter 2009 ‘Earmuff Maniac’ page, there is an embedded MP4 video that shows the look book for that season.

Rollover Image

On the Autumn Winter 2013 ‘Anatomicourture’ page, there is a hover-effect image that displays the front and back of an outfit. On mobile, the hover effect can be achieved by tapping on and off the image.

Extras - JavaScript

JavaScript was used to implement the dynamic navigation menu for both the desktop and mobile sites, the ‘back-to-top’ button, and the hovering image effect. Using JavaScript for these features enhances the fluidity of the site and improves user experience. jQuery was used with all scripts to simplify syntax.

Site Summary

With my site, I highlight one of my favorite brands of clothing. I provide background on the brand itself, it’s creator and designer, and a breakdown of the style codes that are included on garment tags. When choosing what seasons to highlight with my site, I chose ones that were widely celebrated within the Undercover community, and ones that had great standout pieces. By complete coincidence, six out of seven seasons were Autumn Winter ones.

I am still not satisfied with the way the banners at the top of each page turned out. I was having trouble scaling them for mobile when the page was shrunk. I tried background images with text on top, background images with text built in, and just plain images with text on top. I settled on using a background image with a transparent text PNG that would scale with the width of the page. I also wanted to include an interactive section on the Style Codes page that would allow users to enter a style code, and the site would determine the season/line/type automatically. I would have done this using an input box and JavaScript.
